# odin-tic-tac-toe
TOP - Project: Tic Tac Toe
Objectives:
 - Create a basic Tic Tac Toe JavaScript game following instructions at [Project: Tic Tac Toe](https://www.theodinproject.com/lessons/node-path-javascript-tic-tac-toe)
 - Store the gameboard as an array
 - Minimize global code using factories
 - Focus on a functional game in the console first before transitioning to the DOM and HTML/CSS
 - Create an object htat will handle display/DOM logicc
 - Write functions that allow players to add marks to a specific spot
 - Interface utility (player names, start/restart buttons, results display)
 - Have fun!
